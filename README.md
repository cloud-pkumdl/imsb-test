# 🧬 IMSB 结构生物学人格测试

> *I'm a Structural Biologist* — 用结构生物学的方式认识自己

基于 MBTI 范式设计的趣味人格测试，用结构生物学日常场景定义你的科研人格。

## 四大人格维度

| 维度 | 两极 | 含义 |
|------|------|------|
| 🔬 研究尺度 | **E**xpansive / **I**ntensive | 大系统平台 vs 单靶标深挖 |
| 🧩 方法论 | **N**ovel / **S**tructural | 开创新方法新范式 vs 让实验证据说话 |
| ⚡ 驱动力 | **T**heoretical / **F**unctional | 机制闭环物理化学 vs 功能应用转化落地 |
| 🎯 研究风格 | **J**ourneyman / **P**ioneering | 十年深耕长期主义 vs 灵活迭代追逐未知 |

## 双模式

- **🧬 专业版**：实验室场景（EMDB命名/空间群选择/组会报告/PI幻想曲）
- **🌍 生活版**：日常场景（旅行/乐高/自助餐/团建），同样的维度，更轻松的表达
