# 🧬 IMSB 结构生物学人格测试

> *I'm a Structural Biologist* — 用结构生物学的方式认识自己

基于 MBTI 范式设计的趣味人格测试，用结构生物学日常场景定义你的科研人格。

## 四大人格维度

| 维度 | 两极 | 含义 |
|------|------|------|
| 🔬 方法论 | Experimental / Computational | 你更相信实验数据还是计算预测？ |
| 🧩 思维方式 | Reductionist / Integrative | 拆到原子级别还是看大画面？ |
| ⚡ 工作风格 | Static / Dynamic | 追求单一高分辨结构还是构象ensemble？ |
| 🎯 驱动力 | Mechanism / Application | 搞清楚"为什么"还是做出"能用的"？ |

## 双模式

- **🧬 专业版**：实验室场景（EMDB命名/空间群选择/组会报告/PI幻想曲）
- **🌍 生活版**：日常场景（旅行/乐高/自助餐/团建），同样的维度，更轻松的表达
